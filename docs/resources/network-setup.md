# Network Setup

1. Power up the UniFi Express (USB-C power supply)
2. Connect its WAN port (the ethernet port with the globe beneath it) to a wired Ethernet port
3. Connect to the Robots101 network once it finishes booting
4. Navigate to [https://ias.qut.edu.au/login](https://ias.qut.edu.au/login)
5. Allow the popup (this will grant the whole network internet access)

Optionally, to bring up the network console:

1. Connect to [https://192.168.1.1](https://192.168.1.1)
2. Ignore your browser's warnings and continue anyway
3. Enter the password (stored in club Bitwarden)
4. Navigate to the Network tab (left button on the top bar)
5. Navigate to the Client Devices section (the phone/laptop icon on the left)
